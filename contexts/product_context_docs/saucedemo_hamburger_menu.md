# SauceDemo / Swag Labs — Hamburger Menu Documentation

## Overview

The **Hamburger Menu** provides navigation and utility options accessible from the top-left corner of most pages.

## Menu Items

| Menu Option         | Behavior                                                       |
| ------------------- | -------------------------------------------------------------- |
| **All Items**       | Redirects to `/inventory.html`.                                |
| **About**           | Opens external link to [saucelabs.com](https://saucelabs.com). |
| **Logout**          | Ends session → redirects to login page.                        |
| **Reset App State** | Clears cart, resets product buttons, restores default state.   |

## UX Behavior

* Menu opens as a slide-out panel overlay from left.
* Clicking outside or selecting an item closes the menu.
* The menu is consistent across all logged-in pages.
